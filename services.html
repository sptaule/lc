services

Il est important d'avoir une présence sur internet.

Vous est-il déjà arrivé de ne pas trouver les horaires d'un magasin,
de ne pas avoir accès aux annonces importantes d'un commerce,
ou bien d'atterir sur un site et de ne même pas savoir de quoi il s'agit ?
c'est parfois ennuyant !

Je réalise des sites web pour les petites et moyennes entreprises, associations, clubs, etc.

Il peut s'agir :

  • - d'un site vitrine qui a pour but de vous présenter ou bien de présenter votre activité.
  • - d'un site fonctionnel relatif à l'activité de l'entreprise permettant de gérer un ensemble d'utilisateurs, en leur mettant à disposition des services.
  • - d'un site ecommerce pour promouvoir vos produits et les mettre à la vente.

Le site vitrine

Il vous présente ou bien présente votre activité, vos services.

Qui que vous soyez, peu importe la nature de votre activité, il est vivement recommandé d'être visible sur internet.

Comptez un mimimum de 500€ pour un site vitrine.

La prestation comprend généralement les points suivants :
  • - 15 pages maximum
  • - Charte graphique personnalisée
  • - Site responsive : optimisé pour les différents supports (PC, tablettes, smartphones)
  • - Indexation et optimisation de votre site sur les moteurs de recherche

Le site fonctionnel

Si vous souhaitez mettre en relation vos utilisateurs ou bien leur proposer des services,
c'est le type de site qu'il vous faut.

Des exemples de sites fonctionnels connus et largement utilisés : Facebook, Leboncoin, Airbnb...

Comptez un mimimum de 1000€ pour un site fonctionnel.

La prestation comprend généralement les points suivants :
  • - Nombre de pages illimité
  • - Charte graphique personnalisée
  • - Site responsive : optimisé pour les différents supports (PC, tablettes, smartphones)
  • - Indexation et optimisation de votre site sur les moteurs de recherche

Le site ecommerce

Vous voulez vendre vos produits ou services ?
Vous avez déjà une boutique physique et vous souhaitez également être sur internet ?
Ce dont vous avez besoin est un site ecommerce.

Le principe est simple et vous le connaissez sûrement :
un catalogue de produits/services, un panier, puis la validation de la commande.

Comptez un mimimum de 1500€ pour un site ecommerce.

La prestation comprend généralement les points suivants :
  • - Nombre de pages illimité
  • - Charte graphique personnalisée
  • - Variations de produits
  • - Système de paiement
  • - Suivi des commandes
  • - Gestion du site (administration)
  • - Formation (vidéo) sur la partie administration
  • - Site responsive : optimisé pour les différents supports (PC, tablettes, smartphones)
  • - Indexation et optimisation de votre site sur les moteurs de recherche
